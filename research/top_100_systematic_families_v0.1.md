# LoongEnglish Top 100：中文成体系、英语不成体系的高价值词族

这份表不是最终标准，而是 `LoongEnglish` 当前最值得优先改造的 `100` 组词族清单。

筛选标准只有四条：

1. 中文内部组合稳定
2. 英语对应表达不共形、不透明或历史断裂
3. 改造后有机会做到“一眼能猜”
4. 适合后续继续扩成词库

判定标签说明：

- `历史断裂`：英语用了完全不同的词形
- `同根消失`：英语有词，但失去明显家族关系
- `标签混杂`：英语在多个后缀或多条命名路线之间切换
- `短语补洞`：英语只能靠解释性短语维持表达

## 当前优先级建议

如果要继续推进正式词库，建议优先顺序如下：

1. `肉 / 奶 / 医 / 费 / 证 / 卡 / 网`
2. `学 / 课 / 房 / 室 / 车 / 线 / 运`
3. `法 / 案 / 诉 / 病 / 药 / 诊`
4. `亲属细分 / 相对时间 / 星期编号 / 月份编号`

原因很简单：

- 第一组最容易被人第一眼读懂
- 第二组最容易扩成场景词库
- 第三组专业性高，适合做行业词表
- 第四组价值很大，但需要更强的推广策略
